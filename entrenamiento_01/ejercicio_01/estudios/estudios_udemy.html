Curso de Desarrollo Web Completo

Institución:

Udemy

Curso enfocado en el desarrollo de aplicaciones web desde cero, abarcando tanto frontend como backend. Incluye la creación de proyectos prácticos aplicando tecnologías modernas y buenas prácticas de programación.

Competencias adquiridas

  • Desarrollo de sitios web con HTML, CSS y JavaScript
  • Introducción a frameworks frontend (React)
  • Creación de servidores con Node.js y Express
  • Manejo de bases de datos (MongoDB / SQL)
  • Consumo y desarrollo de APIs REST
  • Uso de Git y GitHub para control de versiones
  • Despliegue básico de aplicaciones web

Periodo
